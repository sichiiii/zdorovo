# Zdorovo 0.1.1

This release polishes the desktop experience and opens the project to the
public with a new English project page.

Highlights:

- full GNOME notification text, application icon and expanded banners;
- a thinner daily-chart scrollbar and clearer values above hourly bars;
- compact habit, wellbeing and application cards without trailing space;
- rounded status banners and a clearer Overview navigation label;
- consistent turquoise switches in the light and dark themes;
- six real interface screenshots generated from a private demo profile;
- a completely rewritten English README.

The Debian package is tested on Ubuntu 24.04 LTS and Debian 13. The active-app
tracking extension supports GNOME 46–50.

Install or upgrade with:

```bash
sudo apt install ./zdorovo_0.1.1_all.deb
```

Zdorovo stores its settings, analytics and achievements locally. It is a
work-rhythm tool, not a medical device.
