# Zdorovo 0.1.5

This release improves calendar ranges, compact-window layouts and achievement progression.

Highlights:

- fills every day between the selected start and end dates;
- uses a continuous turquoise range instead of a faint translucent middle;
- keeps rounded corners only on the first and last days;
- provides the same clear contrast in light and dark themes;
- reflows Overview, Breathing, Habits, Analytics, Settings and Achievements at narrow widths;
- replaces oversized scrollbars with slim application-owned styling;
- removes trailing space from breathing instructions, presets and metric cards;
- expands every achievement series to eight levels, for 56 emblems without empty grid cells;
- keeps every level visible by switching directly between four- and two-column grids;
- adds a configurable 20–120 second duration for Eye rest;
- writes achievement notifications as clear sentences in both interface languages;
- keeps the previous upper goals while adding useful milestones between them.

Install or upgrade with:

```bash
sudo apt install ./zdorovo_0.1.5_all.deb
```
