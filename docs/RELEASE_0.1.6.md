# Zdorovo 0.1.6

This release adds complete accent palettes and tightens the relationship between activity tracking, wellbeing and appearance settings.

Highlights:

- choose teal, burgundy or gray without losing the light, dark or automatic theme mode;
- apply the selected accent consistently to controls, cards, SVG icons, charts, notifications and breathing visuals;
- choose the accent from a dedicated Settings card with a palette emblem and a colour marker for every option;
- keep appearance mode and automatic switch times together in a responsive card;
- use calm, even dropdown borders without a bright upper edge in dark mode;
- hide the daily wellbeing card, or keep it visible while disabling only its reminders;
- distinguish dropdown controls more clearly on light backgrounds;
- remove space previously reserved by hidden Overview status messages;
- let compact Behaviour cards use their natural height without blank lower halves;
- count fullscreen films and presentations as screen time even when the mouse remains still;
- carry the new palette and wellbeing choices through JSON export and import.

Install or upgrade with:

```bash
sudo apt install ./zdorovo_0.1.6_all.deb
```
