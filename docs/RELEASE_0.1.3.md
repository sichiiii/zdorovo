# Zdorovo 0.1.3

This release improves media recovery and extends long-term achievement progress.

Highlights:

- resumes only the media players Zdorovo paused, after either Done or Remind later;
- expands the collection from 22 to 38 achievement emblems;
- adds levels V–VII and goals up to 1000 completed activities;
- adds long-term goals for eye rest, movement, breathing, habits and streaks;
- wraps larger achievement series into readable four-column rows;
- preserves existing unlock IDs, exports and imported progress.

Install or upgrade with:

```bash
sudo apt install ./zdorovo_0.1.3_all.deb
```
