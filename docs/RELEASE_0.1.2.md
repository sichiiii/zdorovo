# Zdorovo 0.1.2

This release adds scheduled themes and tightens the main window layout.

Highlights:

- light, dark and automatic appearance modes;
- configurable light and dark switch times in 15-minute steps;
- automatic switching while Zdorovo is running in the background;
- compact vertical scrollbars in light and dark themes;
- equal-height wellbeing and quick-start cards on the Overview page;
- theme settings included in exported backups.

Install or upgrade with:

```bash
sudo apt install ./zdorovo_0.1.2_all.deb
```
