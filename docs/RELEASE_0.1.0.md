# Zdorovo 0.1.0

Первый пакетный релиз Zdorovo — приложения для коротких оздоровительных пауз
во время работы за компьютером.

В релиз вошли:

- настраиваемые паузы для глаз, движения, шеи, плеч, спины и кистей;
- дыхательные сессии с визуальным ритмом и звуковыми подсказками;
- привычки с дневными целями и необязательными напоминаниями;
- 22 достижения в семи многоуровневых сериях;
- локальная аналитика экранного времени по дням, часам и приложениям;
- дневник самочувствия и статистика выполненных упражнений;
- защита от напоминаний при демонстрации экрана и в полноэкранном режиме;
- экспорт и импорт настроек, аналитики и достижений;
- интерфейс на русском и английском языках.

Пакет проверен на Ubuntu 24.04 LTS и Debian 13. Расширение отслеживания
активного приложения рассчитано на GNOME 46–50.

Установка:

```bash
sudo apt install ./zdorovo_0.1.0_all.deb
```

SHA-256:

```text
9e7480b9580c4ff6c1752621db023e99dcab8287aa01a6e1461dcea7c94851c6
```

После установки откройте Zdorovo из списка приложений. Фоновые службы будут
автоматически запускаться вместе с пользовательской сессией.

Zdorovo не является медицинским устройством и не заменяет консультацию врача.
