# Contributing

1. Создайте отдельную ветку для изменения.
2. Не добавляйте пользовательские базы, конфиги, дампы и снимки экрана с личными данными.
3. Перед отправкой запустите тесты и Ruff:

   ```bash
   python3 -m unittest discover -s tests -v
   ruff check .
   ruff format --check .
   ```

4. Для изменений интерфейса проверьте английский и русский языки, светлую и
   тёмную темы, а также окно напоминания.
5. В описании pull request укажите, что изменилось и как это проверялось.
